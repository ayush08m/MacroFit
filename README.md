# 🍽️ Calorie Tracker App

A modern **React Native** calorie tracking application built with **Expo** that helps users monitor their daily nutrition. Users can easily add meals, track calories and macronutrients, view meal history, and maintain healthy eating habits through a clean and intuitive interface.

---

## ✨ Features

- 🍽️ Add meals with calorie and macro information
- 📊 Track daily totals for calories, protein, carbohydrates, and fat
- 📝 View all meals or recent meals on the home screen
- 📤 Share and copy meal information
- 📳 Haptic feedback for a better user experience
- 🔔 Daily reminder notifications (iOS)
- 💾 Store data locally using AsyncStorage
- 📱 Modern tab-based navigation
- ⚡ Fast, responsive, and clean UI

---

## 🛠️ Tech Stack

- React Native
- Expo (SDK 55)
- TypeScript
- Expo Router (File-based Routing)
- AsyncStorage
- Expo Notifications
- Expo Haptics

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/ayush08m/MacroFit.git
cd calorie-tracker
```

### Install Dependencies

```bash
npm install
```

### Run the App

```bash
npx expo start
```
