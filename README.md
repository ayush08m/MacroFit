A modern React Native calorie tracking application built with Expo that helps users monitor their daily nutrition. Easily log meals, track calories and macronutrients, and stay consistent with healthy eating through a clean, intuitive interface.

✨ Features
Add meals with calorie, protein, carbohydrates, and fat information
Track daily nutrition totals automatically
View all meals or recent meals from the home screen
Share or copy meal details
Haptic feedback for a better user experience
Daily reminder notifications (iOS)
Local data persistence using AsyncStorage
Modern tab-based navigation
Responsive and clean UI built with TypeScript
🛠️ Tech Stack
React Native
Expo (SDK 55)
TypeScript
Expo Router (File-based Routing)
AsyncStorage
Expo Notifications
Expo Haptics
